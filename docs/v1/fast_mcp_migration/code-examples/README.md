# Code Examples

This directory contains reference code examples for the FastMCP migration.

## Examples

- **[fastmcp-server.py](fastmcp-server.py)** - FastMCP server implementation with middleware
- **[tool-example.py](tool-example.py)** - Example tool implementation
- **[main-stdio.py](main-stdio.py)** - Main entry point for stdio transport
- **[main-http.py](main-http.py)** - Main entry point for HTTP transport with auto-reload
- **[validation-scripts.md](validation-scripts.md)** - Validation script examples

## Usage

These examples demonstrate the recommended patterns for:
- Dependency injection via middleware
- Tool implementation with context state
- Lifespan management
- HTTP and stdio transports

See the individual phase documentation for context on when to use each pattern.
